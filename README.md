# Homebrew Tap for LightWave Media

This is the official Homebrew tap for LightWave CLI tools.

## Installation

```bash
brew tap lightwave-media/tap
brew install lw
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `lw` | Unified CLI for LightWave Media (Go/TypeScript/Python) |

## Updates

Formulae are automatically updated via GoReleaser when new releases are published.
